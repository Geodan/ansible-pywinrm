PyWinRM Role
=========

Installs PyWinRM (Python Windows Remote Management) for administrating Windows systems from Ansible.

Requirements
------------

* Python
* Python-Pip

Role Variables
--------------

* pywinrm_version: Specific verison of PyWinRM to be installed. By default the latest will be installed.
